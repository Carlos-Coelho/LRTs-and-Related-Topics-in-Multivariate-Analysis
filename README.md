# LRTs-and-Related-Topics-in-Multivariate-Analysis
folder with Mathematica, R and MAXIMA modules to implement distributions mentioned in the book ``Likelihood Ratio Tests and Related Topics in Multivariate Analysis -- Selected Contributions of Carlos A. Coelho'' (https://link.springer.com/book/9783031680076).

It has the modules referred in Chapter-8 (A General Near-Exact Distribution Theory for the Most Common Likelihood Ratio Test Statistics Used in Multivariate Analysis), Chapter-10 (The Exact and Near-Exact Distributions of the Main Likelihood Ratio Test Statistics Used in the Complex Multivariate Normal Setting) and Chapter-12 (On the Distribution of Linear Combinations of Chi-Square Random Variables).
